The Guitar Reference - An ST Guitar Tool

 

The following links are to screenshots of the Guitar Tool in action. These are probably bigger than simply downloading the tool and giving it a whirl but hey, if you wanna look, you go right ahead, that’s what this page is here for

 

.

The Main Fretboard

This picture shows the main work screen. It is the guitar fretboard with the scale. This can be resized and notes can be displayed either as text, as squares for speedy drawing or as circles. Notes can be added and deleted either by clicking on the fretboard or from the pull down menus.

The current scale name is shown in the information bar of the window. This can be changed to show the scale name and the key, or just the scale name, or just the key.

The key can be changed from the pull down menu. See what it did to the Pentatonic Blues Scale .

 

The Tuner

The program comes with a comprehensive tuner . Notes can be played through midi or soundchip and a number of tunings can be stored in memory. The selection of notes comes with an easy to use octave selection dialogue, so that no knowledge of music theory is necessary.

 

Other Pics

The following pics are a bit less interesting and detail dialogues which come up in the program but if you want to see them....go ahead, make your day.

 

The Display Options

This picture shows the display options which are available. The fret numbering can be changed. The display of notes as text, circles or squares for speedy drawing can be selected. The size of the frets can be altered from way-tooo-titchy-tinny-small to ooops-far-to-large-indeed. The relative scale can be selected.

 

Printing

At the moment printing is done without GDOS so the printing dialogue is a bit daunting, but you can print to disk.

 

Tutor

The program is useful for learning the fretboard. By clicking on any note you can get a display of information about that note .

 

Setup

The Guitar Reg uses a comprehensive defaults setup which allows your favourite defaults to be loaded in when the program is run.

 

Helpful

The program will also tell you what the on-screen symbols mean when you press help.
